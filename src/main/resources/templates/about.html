Описание приложения

Приложение представляет собой систему для проведения (кибер)спортивных мероприятий

Список возможностей

  • Создание команды с логотипом
  • Добавление состава команды
  • Вывод списка команд со статистикой
  • Создание матча с двумя участниками
  • Настройка счета участников матча
  • Работа с данными исходя из роли и идентификатора пользователя

Список возможностей администратора

  • Список пользователей
  • Форма редактирования пользователя с настройкой ролей
  • Представление для исключений
  • Ограничение работы с контроллерами по ролям пользователя
